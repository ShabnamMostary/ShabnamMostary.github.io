Garden Planner

A full-stack website to plan a garden. This site allows users to search for plants by name including partial searches and returns detail information about them. Users can create their own accounts, add, or remove plants in their garden. Integrated the website with auth0 authentication platform for secure user management.

Interactive Google map

Designed and developed a dynamic user-friendly map of clinical trials.The interface allows users to view search locations in a google map. Locations in the map are dynamically updated when a user select a trial from the trial list. Upon clicking on specific marker in the map, user will receive details about the location in information box It also supports visiting the trial page from the info box.The project was implemented using react and react-google-maps/api

School

A web interface of an imaginary school to search for faculty and show faculty details using reactJS. The frontend uses RESTful API that accesses information about departments, faculty and students from SQL Database. The API allows search for faculty or students by name, supports adding and removing faculty and students from the database. Applied MVC principles to design and implement this project.
